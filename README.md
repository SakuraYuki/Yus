# .NET Yus Utility

基于.NET 4.0的实用工具集，解决小型项目快速开发的问题，最小侵入性。

## 特色功能

- 丰富的扩展方法，以Yus开头，避免冲突
- 实实在在解决项目中的常见问题，比如时间格式
- 提供License授权检测，文件和字符串的双向加解密模块
